Realistic Swamp Render in Blender

This was done using Quixels Megascans model library and using the "Scatter Objects" addon to scatter the grass. The texturing was procedurally added using Blender's node system and the water texture was blended using a similar technique in the nodes workspace. There was a final composition using the mist pass and color grading along with the "Intel Open Image Denoise" denoiser.
